# LOG2990 Discord Bot

In order for the bot to work, you need to include your credentials in a `.env` file located in the `src/` folder.

The file should look like this:

```
DISCORD_TOKEN=XXXXXXXXXXXXXXXXXXXXXXXX
SERVER_ID=
ID_ROLE_PROFESSEUR=
ID_ROLE_CHARGE=
ID_ROLE_REPETITEUR=
PREFIX=!
CLASS=LOG2990
```
